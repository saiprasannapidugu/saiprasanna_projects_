# 🌍 AI Travel Planner by Mood & Budget

## 📌 Project Overview
AI Travel Planner is a web-based application that helps users plan their trips easily by recommending travel destinations, hotels, restaurants, and tourist attractions based on their **mood and travel budget**.

The system analyzes user inputs and generates a **personalized travel itinerary** with suitable suggestions for accommodation, food, and places to visit.

This project aims to simplify trip planning and provide users with an organized travel plan in one place.

---

## ❗ Problem Statement
Planning a trip can be time-consuming because travelers often need to search different websites for destinations, hotels, restaurants, and attractions. It also becomes difficult to balance these choices with a limited budget.

This project solves this problem by providing **automatic travel recommendations and a structured itinerary**.

---

## 💡 Solution
The AI Travel Planner collects information from the user such as:

- Mood (Adventure, Relaxation, Cultural, etc.)
- Travel budget
- Preferred location

Using this information, the system selects appropriate destinations from a dataset and generates **hotel, restaurant, and tourist attraction recommendations** along with a **daily travel schedule**.

---

## ⚙️ Technologies Used
- 🐍 **Python** – Backend logic
- 🌐 **Flask** – Web application framework
- 🧾 **HTML** – Web page structure
- 🎨 **CSS** – User interface styling
- 📂 **JSON** – Dataset storage for travel destinations

---

## ✨ Features
✔ Travel suggestions based on user mood  
✔ Budget-friendly hotel recommendations  
✔ Restaurant suggestions for breakfast, lunch, and dinner  
✔ Tourist attraction recommendations for day and night  
✔ Automatic **day-wise itinerary planning**  
✔ Simple and user-friendly interface  

---

## 🧠 How the System Works
1️⃣ The user enters mood, location, and travel budget.  
2️⃣ The system processes these inputs.  
3️⃣ Destination data is fetched from the dataset.  
4️⃣ The application recommends hotels, restaurants, and attractions.  
5️⃣ A **complete travel itinerary** is generated for the user.
